\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_pipeline_service_1_1_pipeline_service_manager}{Pipeline\+Service\+Manager}} is in charge of delivering Pipeline\+State\+Resources, Shaders, Root signature objects, and any other graphics-\/pipeline-\/related tools needed by the \mbox{\hyperlink{class_main_application_orchestrator}{Main\+Application\+Orchestrator}} or a renderer to execute a draw function on the G\+PU 

This class requires resources allocated by the \mbox{\hyperlink{class_direct_x_manager}{Direct\+X\+Manager}} and passed by the \mbox{\hyperlink{class_main_application_orchestrator}{Main\+Application\+Orchestrator}} to initialize Pipeline\+State\+Resources, Shader\+Resources, Input layouts, Rasterization stages, stenciling pipelines and more.
