\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_direct_x_manager}{Direct\+X\+Manager}} is one of the primary services exposed to the \mbox{\hyperlink{class_main_application_orchestrator}{Main\+Application\+Orchestrator}}. It handles the creation of the device for interfacing with the G\+PU, can alter Render\+Targets in response to changes in the \mbox{\hyperlink{class_viewport_window}{Viewport\+Window}}, and manages the creation, allocation, and submission of command lists. 

\mbox{\hyperlink{class_direct_x_manager}{Direct\+X\+Manager}} is also the final stop point for synchronization with the G\+PU.
