Detailed Description

The PipelineServiceManager is in charge of delivering PipelineStateResources, Shaders, Root signature objects, and any other graphics-pipeline-related tools needed by the MainApplicationOrchestrator or a renderer to execute a draw function on the GPU

This class requires resources allocated by the DirectXManager and passed by the MainApplicationOrchestrator to initialize PipelineStateResources, ShaderResources, Input layouts, Rasterization stages, stenciling pipelines and more.
