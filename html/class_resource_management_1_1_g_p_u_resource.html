Detailed Description

template<typename T>
class ResourceManagement::GPUResource< T >

Runtime resources are non-fetchable resources. They are meant to be constant buffers or other resources that are not shared and not serialized. They are built at runtime and not loaded from a data segment or stored as a key in the database. The owner is responsible for calling releaseResources with a reference to the runtime subresource to remove it.
